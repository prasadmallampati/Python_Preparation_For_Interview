# Introduction to Python

## What is Python?

Python is a **high-level, interpreted, general-purpose programming language** known for its simple syntax, readability, and versatility. It allows developers to write clean and maintainable code while supporting multiple programming paradigms, including object-oriented, procedural, and functional programming. :contentReference[oaicite:0]{index=0}

# History of Python

Python was conceived in the **late 1980s** and development began in **December 1989**. The first public release (**Python 0.9.0**) was made in **February 1991**. It has since become one of the world's most widely used programming languages. :contentReference[oaicite:1]{index=1}

---

# Who Developed Python?

Python was created by **Guido van Rossum**, a Dutch software engineer.

He designed Python with the goal of making programming:

- Easy to learn
- Easy to read
- Easy to maintain
- Highly productive

For many years, Guido served as Python's **Benevolent Dictator For Life (BDFL)** before the language transitioned to governance by the Python Steering Council. :contentReference[oaicite:2]{index=2}

---

# Where Was Python Developed?

Python was developed at the **Centrum Wiskunde & Informatica (CWI)** in Amsterdam, Netherlands.

Initially, it started as a hobby project during the Christmas holidays while Guido was working at CWI. :contentReference[oaicite:3]{index=3}

---

# Why Was Python Created?

Python was created to overcome the limitations of the ABC programming language and to provide a language that was:

- Simple
- Readable
- Powerful
- Extensible
- Portable
- Easy for beginners

The primary focus was improving developer productivity by reducing code complexity. :contentReference[oaicite:4]{index=4}

# Why is it Called Python?

The name **Python** is **not** derived from the snake.

It comes from the British comedy television show:

**Monty Python's Flying Circus**

Guido van Rossum wanted a short, unique, and memorable name for the language. :contentReference[oaicite:5]{index=5}

# Python is Case Sensitive

Python is a **case-sensitive programming language**.

Example:

```python
name = "Prasad"
Name = "Mallampati"

print(name)
print(Name)
```

Output:

```
Prasad
Mallampati
```

Here, `name` and `Name` are treated as two different variables.

---

# Is Python Compiled or Interpreted?

Python is primarily an **interpreted language**.

Execution flow:

```
Python Code (.py)
        ↓
Python Interpreter
        ↓
Bytecode (.pyc)
        ↓
Python Virtual Machine (PVM)
        ↓
Output
```
